# Laboratorio Constraint Layouts

## 1. Objetivo

Aplicar el uso de restricciones (ConstraintLayout) para la construcción de interfaces de usuario complejas, mediante el desarrollo de formularios funcionales en Android.

---

## 2. Descripción del Proyecto

Este laboratorio implementa dos formularios principales utilizando exclusivamente `ConstraintLayout`:

1. Formulario de envío de correo electrónico.
2. Formulario de registro de solicitud de compra de combustible.

Ambos formularios aplican:

- Restricciones horizontales y verticales.
- Match Constraints (0dp).
- Márgenes.
- Alineaciones.
- Componentes dinámicos.
- Validación de datos.
- Manejo de eventos.
- Navegación entre Activities.

---

## 3. Tecnologías Utilizadas

- Java
- Android SDK
- ConstraintLayout
- Intents explícitos e implícitos
- Spinner
- ArrayAdapter
- ArrayList
- Toast
- StringBuilder
